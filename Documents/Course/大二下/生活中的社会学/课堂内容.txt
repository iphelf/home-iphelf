# 第二讲 社会学研究方法

- 教材

  - 社会研究方法——艾尔
  - 社会调查方法——风笑天

- 大致分为两类：定性研究（Qualitative research）和定量研究（Quantitative research）

- 定量社会调查的例子（人体一致性）

  - 国家人口普查
  - 美国总统大选的民意调查

- 定量研究的概念

  - 确定事物某方面**量的规定性**的科学研究，是将问题与现象用数量来表示，进而去分析、考验、解释，从而获得意义的研究方法和过程
  - 定量，就是以数字化符号为基础去测量。通过对研究对象的特征按某种标准作量的比较来测定对特征数值。由于其目的是对事物及其运动的量的属性作出回答，帮名定量研究。

- 为什么使用定量研究？

  - 研究的社会现象可以测量
  - 可以用统计数据来回答研究问题
  - 可以设计假设检验
  - 在实际生活中，人类社会不是实验室，普查很困难，所以抽样定量研究使用较多。（随机抽样）

- 用中国人口红利（干活的比多于需要被抚养的人）问题说事

  - 目前，中国社会呈现未富先老迹象显现，经济发展压力增强
  - 老龄化（2010:12.5%->2050:30%）

- 定量研究的步骤

  1. 选择研究问题
     - 客观/主观、唯名/唯实、历史/现在...
  2. 回顾相关文献与理论观点
     - 文献综述
       - 发散性列关键词
       - 范性搜索
       - 质量性筛选
       - 相关性精选（浓缩）
       - 综述、回顾
       - 扩展
  3. 建立假设
  4. 选择样本
     - Gallup
  5. 设计调查问卷、建立量尺及指标
  6. 收集、分析资料
  7. 提出结论

- 定性研究

  <!--第四次提问-->

  也称质性研究，是以研究者本人作为研究工具，**在自然情境下**采用多种资料收集方法对社会现象进行整体性探究，使用归纳法分析资料和形成理论，通过与研究对象互动对其行为和意义建构获得解释性理解的一种活动。

  <!--第四次提问-->

  定义分解

  1. 研究环境：在自然环境，社会不是实验室
  2. 研究者的角色：研究者本人是研究工具，是个倾听、思考、观察、挖掘、互动的长期研究，研究者本人对研究的实施很重要。
  3. 收集资料的方法：采用多种方法，如访谈、观察、实物分析等收集资料。
  4. 结论或理论的形成方式：归纳法（敢于提出质疑）、在资料的基础上提升出分析类别和理论。
  5. 理解的视角：研究者与被研究者之间的互动理解被研究者的行为及其意义解释。
  6. 研究者与被研究者的关系：互动的关系，在研究中要考虑研究者个人及其与被研究者的关系对研究的影响，要反思有关的伦理道德问题和权力关系（1. 知情同意书；2. 尊重被研究者：平等对待、被研究者要获益等；3. 保护被研究者的隐私；4. 对方有随时退出的权力）。

- 为什么要定性研究？<!--第四次提问-->

  1. 无法用问卷调查的方法。“真实”依赖被研究者的“全面主诉”，如个人成长的过程。
  2. 无法转化为问卷所必需的指标与变量。如想法、阐释、情绪等。
  3. 已有的统计结果不足以说明问题。如农民工都在找小姐（一个研究结果以偏概全，就会污蔑一个人群）。
  4. 预计可能遇到新的未知的情况。定量研究只能“查明”，不能“发现”。

- 例举几个定性研究

  - 磨憨边境的老挝长途卡车司机
  - 普洱的傣族跨国婚姻
  - 金三角的毒品
    - 老挝毒品
    - 缅甸一侧
    - 忠告
      - 不要轻易饮食陌生环境里的任何东西

- 定性研究设计的主要步骤

  - 第一步：界定关注的社会现象
  - 第二步：文献综述（贯穿整个研究）（酒杯图）
  - 第三步：界定社会问题（实际发生的社会问题）
  - 第四步：明确研究问题
  - 第五步：制定研究方法计划（访谈提纲）
  - 第六步：具体工具设计和抽样
  - 第七步：田野工作（实际调查）：资料收集与分析
  - 第八步：资料分析（第七步中调查的补充）
  - 第九步：产出分析报告

- 几个关键步骤

  1. 选题。定性研究关注事件、过程、意义、整体研究。
     - 从现实社会生活中寻找
     - 从个人经历中寻找
     - 从现有文献中寻找
  2. 进入研究现场。如何接触守门人、被研究访者？如何得到真实信息？
  3. 访谈、观察、照片、绘画、影视（取得被访者的同意）等。一对一访谈（设计较好的情况下），焦点小组访谈。
     - 访谈前的准备工作
       1. 确定访谈的时间和地点
       2. 由被访者定，（1）尊重对方；（2）使被访者感到轻松、安全

# 第三讲 文化 — 我们生活中的真真假假

- 我们生活的世界，为何是井然有序？

  不完全是法律规范的作用

  社会上即使是陌生人相互说话时不会鸡同鸭讲，似乎有着“隐形的”规矩让我们依循

- 这是因为我们拥有共同“文化”的缘故

  文化让社会成员共享同一价值、生活经验，同时造就规范来惩罚失序者。

## 文化的定义<!--第五次提问-->

- 文化其实代表着人类社会的各种生活方式与习惯，包括三类：

  - 物质文化，是我们日常生活中物质与科技的层面，如食物、衣服、房屋
  - 认知文化，使用物质的方法，如习俗、语言沟通、言行举止、信仰价值、哲学等
  - 规范文化，如礼仪规范、民俗乡约、法律法规等

  简言之，文化就是人类社会的生活方式与习惯

  > 文化是人类建构自己的一种意义体系

  通常，认知文化比物质文化更不容易改变，也就是在社会发展过程中，物质文化总是先于认知文化，如工业革命之后，资本主义经济模式才逐渐形成。

  美国社会学家奥格本使用文化滞后（Culture Lag）一词，来指认知文化对其所处的物质环境尚未完全适应的滞后时期。如现代城镇化过程中的价值冲突问题。

## 文化的特点<!--第五次提问-->

1. 超个性。文化是由人类的群体活动体现的，并为人类各群体的成员所共同享有的；如企业文化、民族文化等。

   > 本体论的转变

2. 传承性。文化可后天习得，代代传承下去；如中医、文字等。

3. 整合性。文化不是各种事物简单的组合，而是由多元的文化元素互相关联而形成的完整体系；如中国的新年、端午节等等。

4. 文化是以象征符号为基础的~~，并以此作为一种标志，与动物区别开来~~；货币象征着财富、皇帝的玉玺等。

5. 民族性。文化由一个个民族享有，从而也是各民族借以区别的依据；如傣族、彝族、回族、汉族、藏放的区别等。

### - 规范

规范（Norm）社会所坚持的行为准则

规范本身必须是该社会里的多数人所能接受且了解的。

如在图书馆看书、在电影院看电影，大家需要保持安静。

### 规范的种类<!--第六次提问-->

- 正式规范（外对内的控制）：有文字记载，且对违反者有严格的处罚规定。如法律是国家强制的社会规范，还有大学有必修课、篮球比赛有规则等。

- 非正式规范（内对外的要求）：没有明文记载，但却是为社会大多数人接受。如公交车上年轻人给老人让座、拾金不昧等。

依照对社会的重要性可分为：

- 民德（Mores）：指与一个社会福祉息息相关的规范，这些规范通常包含一个族群最珍视的原则。如忠于国家、杀人偿命、欠债还钱等。
- 民俗（Folkways）：指导我们日常生活行为的规范。如包办婚姻——自由恋爱、拾金不昧、尊老爱幼等。

### 文化与主导意识形态

功能理论学派：一个文化特征或习俗能够持续存在，必然是因为它能够提供社会所需的功能，或能够维持社会稳定及共识。

### 亚文化定义（1）

亚文化指的是社会某一部分人有着与主流社会不同的民德、民俗及价值观。在某种程度上，亚文化可以被视为在主流社会中存在的一个子文化。

如乡村文化、朋克文化、二次元文化、嬉皮士文化等。

农村与城市文化的对比：

1. 俺们刚吃上肉，你们又吃菜了。
2. 俺们刚娶上媳妇，你们又独身了。
3. 俺们刚存上点钱，你们又开始买保险了。
4. ...

### 亚文化定义（2）

“亚文化”既具有被主流社会所划定、又具有群体自我认同的特征：当社会中的某一群体形成了一种区别于占主导地位的文化特征、具有了其他一些群体所不具备的文化要素的生活方式时，这种群体文化便被称为“亚文化”。

农民工文化——听一首歌

### 亚文化特征（1）

1. 与占主导地位的文化相对、相区别。

2. 是某一群体共享的文化，可以被视为主流社会中的一个子文化。

   //佛系群体：不那么为难自己

3. 有一套自己的隐语——专属语言。如：跑酷、跆拳道、街舞、同性恋等。

4. 在这种亚文化下有着自己的规范（种类、变迁）成为行为准则。如日本“宅文化”是随着动漫和计算机游戏的出现而诞生的。

5. 亚文化群体有着自己特有的价值观。如同性恋群体，具有独特的行为规范和方式。一直有争论，从被污名正逐步被更多的人接受。

6. 在这样的价值观和规范下，有着相应的奖惩规则特点。如写的书被认可。

功能理论学派认为，亚文化存在的原因是社会环境本身就具有差异，而亚文化正好印证了这种合理的多样性。

### 我消费我存在

消费是现代人最重要的文化现象之一。//让・波德里亚

回顾工业时代，购买额外物品是奢侈的行为，但工业化之后，消费都成为了以消费而乐的“败家族”。

购物不再是满足需求的行为，而是一个充满乐趣的消费文化。

“物”仿佛成为了一种有生命的，并可以自行衍生的体系，让・波德里亚称之为“物体系”

### 理性地使用“物”

马克思认为人类和自然互动时，人类总会因生存所需而创造出一些物品，这些物品的制造是无法与劳动者分享的，因为它被劳动者所控制、制造。——**使用价值**

马克思认为当劳动者为了某一个资本目的而制造物品时，物的存在目的从此与劳动分离，物品不是为劳动者所用，而是进入市场被交换，称作“**交换价值**”

但在现在社会，很多人似乎忘记了自己是商品的生产者、商品价值的真正赋予者。

反而相信价值是事物本身的自然出现，或者是市场赋予了商品价值，成为物的被统治者。总之人们对物品的控制力越来越弱。

物品的符号价值超越了物品的使用价值和交换价值。

我们的消费目的通常不是在于物品是否有用，也不在乎市场价格，而是享受一种气氛，物品是否“有用”并不是那么重要。如：去咖啡馆“喝咖啡”；买名包。

物：“粒粒皆辛苦”的使用价值→信用卡“借钱享受”的消费时代（符号价值）。

转变：满足需要的消费→（由主观建构起来的）欲望消费

这个时代，生产方式发展的动力，已不再完全属于理性的经济、政治领域，**文化领域**成为社会发展的新动力。

如中国房价的疯狂增长未必是实际需求的结果。文化赋予“房”的符号价值（家，根，归属，（政策）户口，救医），对房产市场具有更深远的意义。拍卖会。

**物**的这种符号价值一旦得到社会的公认（形成文化），将反过来规定着我们的现代生活（界定我们自己），并形成一个符号系统。

**消费**不仅是物质实践，也不是简单的“富裕”现象，而是一种（建立）关系的主动模式（这不只是人和物品之间的关系，也是人和集体、人和世界之间的关系），它是一种系统性活动的模式，也是一种全面性的回应，在它之上，建立了我们的文化体系的整体。——让・波德里亚

消费成为一种阶层、信任、权利、健康、道德及习惯的礼仪标准等。

现代社会特点：时间加速、空间紧缩→信息爆炸；→事件超载；→意义超载（意义淡化、分散以至缺失）；同质化（人人平等（身份证）→人与人的交互少了→没有存在感、生活意义→生活压力）

### 文化品味的竞争时代

日常生活中的“比较”一词，在社会学家看来是一种“竞争”。

位于社会不同阶级的人们，唯有把别人比下去，自己所处的社会地位或**阶级**才会向上提升。

但是，单以经济层面作为人们彼此竞争的基础，无法全面解释社会阶级间的争斗。

布迪厄认为需要以文化层面的品味竞争来理解。

### 文化爱好与品味

文化爱好、品味、鉴赏能力或情趣是人的一种心态、情感和秉性，具有显著的社会区分功能。

爱好的实践逻辑：**不同于普通的以消耗体力为主的笨重体力劳动**，它是作为美的鉴赏活动，它追求的是摆脱一切利益的某种鉴赏乐趣和精神品味，这是一种高于一般认知活动、伦理活动及其它社会活动的最精致而又复杂的实践。//不可习得性道德

艺术作品的物质性或象征性消费，表现为一种最高形式的悠闲自在状态。

正是在这种身体方面和精神方面的“悠闲自在”的实际双重表现中，展示了一般经济性物质消费所达不到的高雅性，甚至由此可以展示出对纯经济优越地位的鄙视，而达到对经济地位差异的否定性的“超越”效果。

达到与“必然王国”相区别的“自由王国”的间距效果——康德。

文化鉴赏活动中的各种爱好、偏向、品味及乐趣运动，是夹杂着奥妙的心领神会精神活动**实践**，其中交集着历史的、文化的和个人有机体的生命力，综合着社会和鉴赏者的才情和哲思及个人天赋，孕育着无限的理想和情操，表达出发自心灵深处的寄托和期望的张力结构。//布迪厄《文化的区分》

### 文化爱好的权力动作和社会区分

首先是文化爱好者的特殊社会地位，呈现他们的社会资源和才能的掌握程度，也表现在他们文化修养和文化资本的累积状况和社会权力的掌握。

人们通过文化爱好的社会表现，如穿着服饰、文学和艺术鉴赏活动等，有意识或无意识地参与了文化爱好方面的权力竞争，实现了权力争夺和较量。

### 秀出自己最独特的品味

秀异：优异特出。布迪厄认为只要是任何企图通过区分自己与大众品味有所差异，并自认为品味较优秀者，就符合“秀异”的概念。

不过，秀异的行为表现必须在两种社会阶级成员相互较劲下才能凸显。

如在吃、穿、住房、商店、家具、音乐、书籍、喜剧、美术等方面展示自己的消费品味，并以此来衬托自己的身份、地位。

其实质是消费主义制造了身份平等的幻觉。
